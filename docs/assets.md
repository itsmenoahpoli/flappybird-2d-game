# Assets

The following assets have been pre-installed in the `flappybird-2d-game/Assets/Sprites` directory:

- **background.png**: The game background (day theme).
- **bird.png**: The bird character sprite.
- **pipe.png**: The pipe obstacle sprite.
- **base.png**: The ground/floor sprite.

Make sure to assign these sprites to your corresponding GameObjects in the Unity Editor. For audio, you can add your own sound effects to the `Assets/Audio` folder.
